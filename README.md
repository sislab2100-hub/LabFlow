# 🧪 LabFlow

**Sistema de Filtrado de Estudios para Laboratorios Clínicos**

Herramienta diseñada para agilizar el flujo de trabajo en laboratorios de análisis clínicos. Filtra pacientes por tipo de estudio, genera hojas de trabajo e imprime subrogados.

🌐 **[Ver Demo en Vivo](https://sislab2100-hub.github.io/LabFlow/)**

---

## ✨ Características

- 📂 **Carga de archivos Excel** — Lee listas de pacientes desde archivos .xlsx/.xls
- 🔍 **Filtrado por estudios** — Selecciona múltiples estudios y filtra pacientes
- 🕐 **Filtro por turno** — Matutino, Vespertino, Nocturno, Jornada
- 👤 **Perfiles guardados** — Guarda combinaciones de estudios frecuentes
- 📋 **Hoja de Trabajo** — Genera documento para el área de trabajo
- 📄 **Subrogado** — Imprime formato para estudios subrogados
- 🧫 **Contador de tubos** — Cuenta tubos por color automáticamente
- 💾 **Sin instalación** — Funciona 100% en el navegador

---

## 🚀 Cómo Usar

1. Abre la aplicación: [sislab2100-hub.github.io/LabFlow](https://sislab2100-hub.github.io/LabFlow/)
2. Carga tu archivo Excel con la lista de pacientes
3. Selecciona los estudios que deseas filtrar
4. Haz clic en "Filtrar seleccionados"
5. Genera tu Hoja de Trabajo o Subrogado

## 📋 Estudios Soportados

La aplicación incluye más de 40 estudios clínicos organizados por categoría:

- **Serología** — VIH, VHB, VHC, VDRL, ASO, FR, PCR, etc.
- **Hematología** — BH, Grupo y Rh, Coombs, etc.
- **Química Clínica** — Glucosa, Perfil Lipídico, PFH, Electrolitos, etc.
- **Hormonas** — Perfil Tiroideo, Estradiol, FSH, LH, etc.
- **Orina** — EGO, Microalbuminuria, etc.
- **Especiales** — Gasometría, HbA1c, Hemocultivo, etc.
